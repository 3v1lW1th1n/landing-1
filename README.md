# JSDaddy - landing page

#### Installation
```
## Install dart using brew to your local machine
brew tap dart-lang/dart
brew install dart --devel
```

- > pub get

#### Development
- > dart --enable-vm-service bin/dev.dart
- > npm run start:dev
- > npm run build:js:w

#### Start server on the 3000 port
- > http://localhost:3000/
